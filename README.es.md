## Dataset

Se utilizó un dataset de calidad del aire para entrenar un modelo de Machine Learning capaz de predecir si los niveles de contaminación representan condiciones seguras o perjudiciales para la salud.

El conjunto de datos incluye mediciones de contaminantes como CO, NO₂, SO₂, O₃, PM2.5 y PM10, variables comúnmente utilizadas para evaluar la calidad del aire.

Este dataset fue seleccionado porque representa un problema real y permite demostrar el flujo completo de desarrollo de un modelo de Machine Learning y su despliegue en una aplicación web.

Nota: La aplicación se encuentra desplegada utilizando el plan gratuito de Render. Debido a las limitaciones de recursos del servicio gratutio, el endpoint puede presentar tiempo de carga elevados o periodos de inactividad.
